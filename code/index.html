Mikado

Mikado is a sweet and nifty way of taking webapps to your desktop. It was made having in mind the mobile version of websites.
Usually this websites are carefully created trying to show the maximum information and functionality using the minimum space.

It allows you as well creating desktop applications using HTML5 technologies. Stop messing with complicated GUI toolkits and start using your well known HTML5 + CSS + Javascript tech.

You can dock your app in the menu / taskbar of your operating system, run it as a window application or create an overlay on top of your desktop.
